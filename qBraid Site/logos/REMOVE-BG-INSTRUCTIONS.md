# Remove Background from Sandy Irani Photo

## Quick Online Methods (No Software Needed):

### 1. **Remove.bg** (Recommended - Takes 5 seconds)
- Visit: https://www.remove.bg/
- Upload `sandy-irani.jpg`
- Download the result (will be PNG with transparency)
- Replace the current file

### 2. **Photopea** (Free Photoshop Alternative)
- Visit: https://www.photopea.com/
- Open `sandy-irani.jpg`
- Use Magic Wand tool to select background
- Delete selection
- File → Export As → PNG (to keep transparency)

### 3. **macOS Preview** (Built-in)
1. Open `sandy-irani.jpg` in Preview
2. Click the Markup toolbar button
3. Select "Instant Alpha" tool
4. Click and drag on the background
5. Press Delete
6. File → Export → Format: PNG

## After Removing Background:

Save the new file as either:
- `sandy-irani.png` (recommended for transparency)
- Or overwrite `sandy-irani.jpg`

Then update the HTML if you changed the extension to PNG:
```html
<img src="logos/sandy-irani.png" alt="Sandy Irani">
```

The circular avatar CSS will handle the rest!
